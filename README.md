## Tap Tempo

![Tap tempo screenshot][0]

A Tap Tempo app written in ReasonML.


### Download
[macOS v0.0.1][1]


### Keyboard shortcuts
  - `ESC` or `q` quit app
  - `SPACE` tap tempo


### Development

```
esy install       # install dependencies
esy run           # build and run the app
esy format        # run refmt on source files

yarn dist         # build macOS package
```
